# Personal Learning — Draw Inspiration

This folder captures what was learned, decided, and discovered while building this project with AI assistance. It's a living reference — not documentation of the app, but documentation of the *process* of making it.

## Files

| File | What it covers |
|------|---------------|
| [01-project-arc.md](./01-project-arc.md) | The shape of the project — how it grew, what got added, what the pivots were |
| [02-technical-decisions.md](./02-technical-decisions.md) | The specific technical problems solved and why certain patterns were chosen |
| [03-prompting-guide.md](./03-prompting-guide.md) | How to work with Claude effectively — what prompt patterns worked, what didn't |
| [04-next-project-checklist.md](./04-next-project-checklist.md) | Practical guide: what to set up at the start of any future project |

## How to use this

Read `01` and `02` to recall the context of specific decisions.
Read `03` before starting a new AI-assisted project — it will save you hours.
Use `04` as a literal checklist on day one of your next project.
